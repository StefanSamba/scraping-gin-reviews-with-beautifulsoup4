# Scraping gin reviews with beautifulsoup4

A simple single-page webscraper with beautifulsoup.
This includes:

- Scraping content of a page
- Extracting divs and list items from the scraped content
- Transforming extracted data into a DataFrame
- Plotting the value counts for the rating

## Usage

Optional:
For running .ipynb in a separate virtual environment:
Create virtual env
`virtualenv venv`

Activate virtual env
`source venv/bin/activate`

Install packages
`pip3 install -r requirements.txt`

Required:
Open `webscraper.ipynb` in Jupyter Notebooks or VS Code (or other tool)
